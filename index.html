perl-dtrace by chrisa

DTrace probes and consumer API for Perl

This is a Perl port of "ruby-dtrace", bindings for the probe and consumer APIs of DTrace. What works: * Declarative provider construction * DOF generation and loading * Firing probes directly, with arguments * Probes exported into application code * Is-enabled checks
